# 金鹰工单KPI - Codex极简风格设计规范

## 设计原则
1. **信息优先** - 数据即UI，减少装饰性元素
2. **密度优先** - 高信息密度，紧凑布局，一屏可见更多内容
3. **克制用色** - 黑白灰为基调，仅状态/数据用彩色
4. **排版清晰** - 字体层级分明，间距精确统一
5. **无边框主义** - 用背景色差异和间距分隔，而非边框

## 色彩体系

### 基础色
- 背景: `#FAFAFA` (页面) / `#FFFFFF` (卡片/面板)
- 文字: `#111111` (主) / `#666666` (次) / `#999999` (辅)
- 分隔: `#E5E5E5` (线) / `#F0F0F0` (区域)

### 功能色 (仅用于状态)
- 成功: `#006400` (深绿，非亮绿)
- 警告: `#B8860B` (深黄)
- 错误: `#8B0000` (深红)
- 信息: `#1A5276` (深蓝)
- 进度: `#333333` (黑)

### 暗色模式
- 背景: `#0A0A0A` / `#141414`
- 文字: `#ECECEC` / `#999999` / `#666666`
- 分隔: `#222222` / `#1A1A1A`

## 字体
```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Noto Sans SC", sans-serif;
```
- 页面标题: 20px / 600
- 区块标题: 13px / 600 / 大写 / letter-spacing: 0.05em
- 数据大数字: 32px / 300 (tabular-nums)
- 正文/表格: 13px / 400
- 辅助说明: 11px / 400 / 灰色

## 间距系统 (4px基准)
- 页面边距: 24px
- 区块间距: 20px
- 区块内边距: 16px
- 表格行高: 36px
- 按钮高度: 32px

## 组件规范

### 导航栏
- 极简顶部条: 44px高, 白色背景, 底部1px #E5E5E5
- Logo: 纯文字 "工单KPI", 无图标
- 导航: 纯文字按钮, 当前项下划线2px #111
- 右侧: 项目选择 + 用户信息 + 同步按钮(文字)

### KPI卡片
- 无阴影, 无圆角(或2px), 白色背景
- 数字: 32px 300, 黑色
- 标签: 11px 灰色大写
- 趋势: 12px, 正绿负红
- 网格: 4列等宽

### 数据表格
- 紧凑行: 36px高
- 表头: 11px 大写 灰色, 无背景, 底部1px线
- 行: 悬停 #FAFAFA
- 状态标签: 小字+颜色点, 非彩色背景块

### 图表
- ECharts 灰色系配色
- 网格线: #F0F0F0
- 数据线: #333 / #666 / #999
- 无3D效果, 无渐变填充

### 表单/筛选器
- 内联筛选: 纯文字链接样式, 选中态加粗+下划线
- 输入框: 底部边框, 无背景色, 无圆角
- 下拉: 极简列表, 无阴影

### 按钮
- 主按钮: #111背景, 白字, 无圆角或2px
- 次按钮: 透明背景, #111边框
- 文字按钮: 纯文字+下划线

### 弹窗
- 居中, 白色背景, 1px #E5E5E5边框
- 无圆角或2px, 无阴影或极浅阴影

## 页面布局

### 整合方案 (6页 → 4页)
1. **概览 (Overview)** = 原驾驶舱 + 计划概览合并
   - 顶部: KPI数字 (4列)
   - 左: 四层级发起表格 + 预警列表(紧凑)
   - 右: 趋势图 + 计划状态(WY/IPMS/设备 迷你状态条)
   
2. **工单 (Tickets)** = 原搜索页增强
   - 筛选条 + 数据表 + 分页
   - 右侧可折叠详情面板
   
3. **人力 (People)** = 原人力页
   - 筛选 + KPI表格 + 详情侧栏
   
4. **系统 (System)** = 原配置+同步合并
   - Tab: 同步 / 人力导入 / 项目映射 / 角色映射 / 负责人

### 驾驶舱信息整合
- 移除计划页的独立入口，将WY/IPMS/设备状态压缩为3个mini status bar嵌入概览页
- 预警清单改为紧凑表格行(非卡片)
- 四层级改为横向表格(非垂直卡片)
